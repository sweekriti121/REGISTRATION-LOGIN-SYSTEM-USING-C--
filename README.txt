REGISTRATION / LOGIN SYSTEM USING C++
Used File Handling / Manipulations / Storing extensively.

Salient Features:
Registration with user id and strong password check for new entries
Login allowed, only if already registered.
Checks for login failure attempts (Limited to 3)
Facility to delete already registered account
Emphasis given on modular approach, readability and maintainability of the program
Enhancements in progress:
Encryption and decryption of password
Maintaining visit count to the registration system
